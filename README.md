# OpenClawKit SwiftUI Companion App

## Overview
A SwiftUI-based companion application for OpenClaw, providing an intuitive setup and configuration experience.

## Features
- Guided setup wizard
- Network configuration
- AI provider integration
- Messaging channel selection
- Secure configuration management

## Requirements
- Xcode 15.0+
- macOS 14.0+
- Swift 5.9+

## Installation
1. Clone the repository
2. Open `SwiftUICompanion.xcodeproj` in Xcode
3. Build and run the project

## Configuration
The app provides a wizard-based approach to configuring OpenClaw, supporting:
- Multiple AI providers
- Various messaging channels
- Secure API key management

## Development
- Follow SwiftUI best practices
- Use MVVM architecture
- Implement comprehensive error handling
